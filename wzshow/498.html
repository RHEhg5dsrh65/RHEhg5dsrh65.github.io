Linux统新手指南，如何在Linux上轻松创建新用户

1740837225    git网址导航    已浏览865次

  1. 登录Linux系统
  2. 查看当前用户
  3. 创建新用户
  4. 验证新用户
  5. 修改用户权限

随着Linux系统的广泛应用，越来越多的用户选择在个人或企业环境中部署Linux操作系统，Linux系统以其稳定性骑士下载站、安全性以及开源特性赢得了广泛的好评，在Linux系统中，创建新用户是日常管理中的一项基本操作，本文将为您详细讲解如何在Linux上创建新用户，助您快速掌握这一技能。

登录Linux系统

您需要登录到Linux系统，如果您是Linux系统的初学者，可以尝试使用虚拟机软件（如VirtualBox、VMware等）安装一个Linux发行版，例如Ubuntu、CentOS等，安装完成后，您可以使用用户名和密码登录到系统。

查看当前用户

在创建新用户之前，我们先来查看一下当前系统中已经存在的用户，在终端中输入以七七下载站下命令：

whoami

该命令会显示当前登录的用户名，如果您的用户名是“linuxuser”，则命令的输出结果将是“linuxuser”。

创建新用户

在Linux系统中，创建新用户可以通过以下几种方式实现：

1. 使用useradd命令

useradd命令是Linux系统中创建新用户的主要命令，以下是一个简单的示例：

sudo useradd -m -d /home/newuser -s /bin/bash newuser

参数解释：

  • -m：创建用户时同时创建用户的家目录。
  • -d：指定用户的主目录路径。
  • -s：指定用户的默认shell。
  • newuser：新创建的用户名。

执行上述命令后，系统会创建一个名为“newuser”的新用户，并为其创建家目录“/home/newuser”，默认shell为“/bin/bash”。

2. 使用vi或nano编辑器

如果您更喜欢使用文本编辑飞飞影视器，可以使用以下命令创建新用户：

sudo vi /etc/passwd

在打开的文件中，找到最后一行，添加以下内容：

newuser:x:1001:1001::/home/newuser:/bin/bash

参数解释：

  • newuser：新创建的用户名。
  • x：密码，这里留空表示密码为空。
  • 1001：用户ID。
  • 1001：组ID。
  • /home/newuser：用户的主目录。
  • /bin/bash：用户的默认shell。

保存并退出编辑器，使用以下命令更新密码：

sudo passwd newuser

系统会提示您输入新密码，并确认密码。

3. 使用图形界面

在Linux系统中，部分发行版提供了图形界面，您可以通过图形界面创建新用户，以Ubuntu为例，在系统设置中找到“用户与群组”选项，点击“添加”按钮，按照提示输入用户名、密码等信息即可。

验证新用户

创建新用户后，您可以使用以下命令进行验证：

su - newuser

该命令会切换到新创建的用户“newuser”下，在提示符下输入命令，即可使用新用户进行操作。

修改用户权限

默认情况下，新创建的用户属于普通用户组，权限较低，如果您需要为用户分配更高的权限，可以使用以下命令：

sudo usermod -aG sudo newuser

该命令将新用户添加到sudoers组，使其具有执行sudo命令的权限。

本文详细介绍了如何在Linux上创建新用户，包括使用useradd命令、编辑器以及图形界面等方法，掌握这些方法，可以帮助您更好地管理Linux系统中的用户，希望本文对您有所帮助！
