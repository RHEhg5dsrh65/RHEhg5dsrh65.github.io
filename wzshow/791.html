什么是GRUB？

GRUB，全称为GRand Unified Bootloader，是一种广泛使用的多系统启动加载器，它负责在计算机启动过程中加载操作系统，支持包括Linux和Windows在内的多种操作系统，GRUB的核心功能是提供一个启动菜单，使用户能够选择启动哪一个操作系统。

开机时GRUB启动菜单的内涵

当您开机时遇到GRUB启动菜单，这通常意味着系统正在搜索可用的启动选项，以下是一些常见的GRUB启动菜单情况：

  • 默认启动项：GRUB菜单中显示的默认启动项是系统配置中设定的默认操作系统，系统启动时会自动加载此选项。
  • 多系统启动：如果您安装了多个操作系统，GRUB菜单会列出所有可用的启动选项，允许用户选择。
  • 修复模式：当系统出现问题时，可以通过GRUB菜单进入修复模式，对系统进行修复。
  • 安全模式：在GRUB菜单中，用户可以选择进入安全模式，以安全模式启动操作系统，便于问题排查。

GRUB启动菜单操作详解

以下是如何在GRUB启动菜单中进行一些常见操作的详细步骤：

1. 修改默认启动项

  1. 当GRUB启动菜单出现时，按下“e”键进入编辑模式。星辰影院
  2. 找到“linux16”或“linux”行，这行代码对应默认启动项。
  3. 按下“e”键编辑该行代码，添加“rw init=/sysroot/bin/bash”以进入单用户模式。
  4. 按下“Ctrl+X”或“F10”键启动修改后的系统。

2. 进入多系统启动

  1. 在GRUB启动菜单出现时，按下“e”键进入编辑模式。
  2. 找到对应操作系统的启动项，按下“e”键编辑。
  3. 修改内核启动参数，例如添加“rw init=/sysroot/bin/bash”以进入单用户模式。
  4. 按下“Ctrl+X”或“F10”键启动修改后的系统。

3. 进入修复模式

  1. 在GRUB启动菜单出现时，按下“e”键进入编辑模式。
  2. 找到“linux”行，按下“e”键编辑。
  3. 修改内核启动参数，例如添加“rw initrd=/path/to/initrd.img-xxx root=/dev/sda1 ro”以进入修复模式。
  4. 按下“Ctrl+X”或“F10”键启动修改后的系统。

4. 进入安全模式

  1. 在GRUB启动菜单出现时，按下“e”键进入编辑模式。
  2. 找到“linux”行，按下“e”键编辑。
  3. 修改内核启动参数，例如添加“single”以进入安全模式。
  4. 按下“Ctrl+X”或“F10”键启动修改后的系统。
